# Area Calculator

This is a Python area calculator for basic geometric shapes.

The program calculates the area of shapes such as triangles, rectangles, squares, and circles. I built this to practice using formulas in code and organizing simple calculations into a working program.

## Concepts Practiced

- Python
- Geometry formulas
- Arithmetic operations
- User input
- Conditional logic

## What I Learned

This project helped me practice translating math formulas into Python and using code to solve simple geometry problems.
